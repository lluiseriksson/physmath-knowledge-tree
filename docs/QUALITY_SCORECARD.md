# Repository-controlled quality scorecard

**Score: 100/100.**

Scope: **repository-controlled quality only**. This score is designed to reach 100 only when all deterministic criteria controlled by the repository pass. It is not a publication probability or a scientific-validity score.

| Criterion | Weight | Earned | Pass |
| --- | ---: | ---: | :---: |
| Canonical graph integrity | 20 | 20 | yes |
| Scoped evidence traceability | 25 | 25 | yes |
| Search and route regressions | 20 | 20 | yes |
| Lean-facing bounded targets | 15 | 15 | yes |
| Automation and research-software documentation | 20 | 20 | yes |

## Explicit exclusions

- scientific truth of heuristic or speculative mechanisms.
- novelty sufficient for publication.
- external adoption, citations or independent replication.
- completed user-study outcomes.
- formalization of the full research programmes.

These exclusions require external review, empirical use or new scientific results and are intentionally not converted into repository points.
