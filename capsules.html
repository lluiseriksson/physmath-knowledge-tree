How research capsules behave

  • The imported dossier fingerprint is verified before use.
  • Run fingerprints, scope links and artifact metadata are checked independently.
  • The browser verifies metadata only; use capsule:verify to hash local artifact files.
  • A ready capsule is a reproducibility handoff, not proof or confidence promotion.
